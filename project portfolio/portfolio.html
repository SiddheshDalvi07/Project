Projects

Project 1 : Login and Register Page

This login and register page features a sleek and modern design, utilizing a two-column layout to display a brand image and form inputs side by side. The page uses the Bootstrap grid system to align the elements and ensure responsiveness across different screen sizes. The login form includes fields for email address and password, as well as a "forgot password?" link for users who need to reset their password. The register form includes fields for name, username, email, and password, as well as a checkbox for agreeing to terms and conditions.
